# Overview

## Purpose

The Enterprise Security Lab models a small-to-medium enterprise network environment designed to demonstrate security-focused network architecture, segmentation, and operational visibility.

The project is intended to serve both as technical documentation and as a portfolio artifact for network and security engineering roles.

## Objectives

The main objectives of the lab are:

- demonstrate enterprise-style network segmentation
- apply firewall-controlled inter-VLAN communication
- separate trusted, untrusted, and management traffic
- document architecture decisions clearly
- support troubleshooting and operational workflows

## Scope

The lab includes documentation for:

- firewall architecture
- managed switching
- wireless access design
- VLAN segmentation
- IP addressing
- network flows
- firewall policy logic
- device inventory
- services and operations
- incident response and controls mapping

## Design Focus

The environment is designed with emphasis on:

- security
- maintainability
- readability
- operational clarity
- enterprise-style documentation

## Intended Audience

This documentation is intended for:

- Network Engineer
- Network Security Engineer
- Senior Infrastructure Engineer
- Network Architect
- technical interview and portfolio review audiences

## Summary

This lab is not only a functional environment, but also a structured documentation project designed to reflect practical enterprise networking and security concepts.
