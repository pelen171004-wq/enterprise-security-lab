The lab implements a zone-based network architecture inspired by enterprise security segmentation models.

Security Zones

  • Users – internal workstation network
  • Guest – untrusted internet-only network
  • Management – administrative access to devices
  • SOC – monitoring and security visibility
  • Servers – internal infrastructure services
  • Backup – backup storage and recovery systems

Traffic between zones is controlled by firewall policies.
