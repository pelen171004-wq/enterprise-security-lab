Internet

Untrusted external network connected through the ISP modem.

  • External threat source
  • North-south traffic entry point
  • Controlled by firewall edge policy

Users VLAN

Standard internal client network for daily endpoint activity.

  • VLAN20
  • Trusted user devices
  • Internet access allowed
  • Management access denied

Guest VLAN

Isolated internet-only access for untrusted endpoints.

  • VLAN60
  • Untrusted device zone
  • Internet access allowed
  • Internal access denied

Security architecture principles:

Management VLAN

Dedicated management plane for infrastructure administration.

  • VLAN30
  • Firewall and switch management
  • Restricted trusted access
  • Protected from users and guests

SOC / Monitoring

Planned monitoring and detection segment for security visibility.

  • VLAN40
  • Logging and SIEM integration
  • Centralized monitoring
  • Security operations tooling

Server Zone

Planned infrastructure and application services segment.

  • VLAN50
  • Internal service hosting
  • Controlled access from trusted zones
  • Firewall policy segmentation

Backup Zone

Planned isolated segment for backup and recovery infrastructure.

  • VLAN70
  • Backup repositories
  • Recovery services
  • Restricted administrative access
