Alert Triage ProcedureON ALERT
1. Classify: CRITICAL (immediate) / HIGH (within 1h) / MEDIUM (within 4h) / LOW (end of day)

2. Investigate: Use diagnose debug flow to trace packet path. Check session table for related connections.

3. Contain: If active threat — add explicit host block policy. Notify via documented escalation matrix.

4. Document: Log in IR case format (see ESL-IR-001 as template). Record MITRE technique if applicable.

5. Close: Remove temporary block if threat resolved. Update IOC feed.
