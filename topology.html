This page documents the physical and logical topology of the Enterprise Security Lab.

The environment is built around a FortiGate firewall acting as the primary security enforcement point and a Cisco CBS250 switch providing VLAN segmentation.

Core Components

  • FortiGate 60E – Next Generation Firewall
  • Cisco CBS250 – Core Switch
  • Vodafone ISP modem – Internet uplink

Network Structure

  • VLAN20 – Users Network
  • VLAN30 – Management Network
  • VLAN60 – Guest Network
