Network Topology

Infrastructure Layout

The lab network is segmented using VLAN architecture with a FortiGate firewall acting as the central security gateway.

  • Internet uplink through ISP modem
  • FortiGate 60E for inter-VLAN routing and policy enforcement
  • Cisco CBS250 for Layer 2 access switching
  • VLAN20 Users, VLAN30 Management and VLAN60 Guest

This topology reflects a simple enterprise security design with clear trust boundaries.
