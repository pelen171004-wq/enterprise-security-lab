Security Objectives

  • Reduce attack surface through segmentation
  • Prevent lateral movement between trust zones
  • Protect management interfaces
  • Allow only required traffic flows
  • Prepare environment for monitoring and incident visibility

Trust Model

Zone Trust Level Purpose
Internet Untrusted External connectivity
Guest VLAN Low Visitor / untrusted endpoints
Users VLAN Medium Internal user devices
Management VLAN High Administrative access

Policy Model

Source Destination Decision Reason
VLAN20 Users WAN ALLOW Business internet access
VLAN60 Guest WAN ALLOW Internet-only guest access
VLAN20 Users VLAN30 Management DENY Management plane protection
VLAN60 Guest VLAN20 Users DENY Prevent lateral movement
VLAN60 Guest VLAN30 Management DENY Block admin access from untrusted network

Security Principles Applied

  • Least privilege
  • Zero Trust inspired segmentation
  • Explicit allow / deny rules
  • Default deny for east-west traffic
  • Administrative isolation
