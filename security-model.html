Security Model

Trust boundaries, threat actors, data flows, and STRIDE threat model for the enterprise lab. Maps identified threats to implemented controls.

Threat Actors

THREATS
External Attacker
THREAT LEVEL: HIGH · Source: WAN
Opportunistic scanners, automated exploit frameworks, nation-state GeoIP sources (RU/CN/KP). Target: any exposed service on WAN.
✓ Control: GeoIP block + WAN deny-all + IPS
Compromised Guest Device
THREAT LEVEL: MEDIUM · Source: VLAN60
Malware-infected IoT or guest device performing lateral movement, scanning internal VLANs, or acting as C2 relay. Scenario documented in IR Case ESL-IR-001.
✓ Control: VLAN60 fully isolated, policy ID 8–10
Insider Threat
THREAT LEVEL: LOW · Source: VLAN20
User on VLAN20 attempting to access management plane or server infrastructure beyond their trust level. Policy enforces least-privilege.
✓ Control: Policy ID 11 (Block-Users-to-Mgmt)
