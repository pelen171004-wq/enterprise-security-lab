Enterprise Security Lab

Enterprise network architecture, VLAN segmentation, firewall security policies and monitoring visibility.

Overview

This project demonstrates an enterprise-style network environment with firewall-controlled segmentation, VLAN architecture, and infrastructure monitoring visibility.

Firewall Traffic Flow

The firewall enforces communication rules between network zones.

  • Users VLAN → Internet (Allow)
  • Users VLAN → Servers (Allow)
  • Guest VLAN → Internet (Allow)
  • Guest VLAN → Internal Network (Deny)
  • Monitoring VLAN → Infrastructure Devices (Allow)
