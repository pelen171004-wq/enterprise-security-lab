Network Attack Surface

Internet Edge

Threat:
  • Port scanning
  • Exploit attempts
  • Brute force
Mitigation:
  • FortiGate firewall
  • NAT policy control
  • Closed inbound ports

Guest Network

Threat:
  • Malicious guest device
  • Lateral movement
Mitigation:
  • Guest VLAN isolation
  • Internet-only policy

User Network

Threat:
  • Compromised endpoint
  • Internal reconnaissance
Mitigation:
  • Inter-VLAN firewall rules
  • Segmentation

Management Plane

Threat:
  • Unauthorized admin access
Mitigation:
  • Management VLAN
  • Restricted access
