CLI — Firewall Policy (Guest-to-WAN example) 
FG60E # config firewall policy
FG60E (policy) # edit 2
    set name          "Guest-to-WAN"
    set srcintf       "vlan60"
    set dstintf       "wan1"
    set srcaddr       "all"
    set dstaddr       "all"
    set action        accept
    set schedule      "always"
    set service       "HTTP" "HTTPS" "DNS"
    set utm-status    enable
    set profile-type  single
    set profile-group "utm-guest"
    set nat           enable
    set logtraffic    all
FG60E (2) # next
FG60E (policy) # end

UTM Security Profiles

UTM
IPS BLOCK
Extended signature set. Blocking mode — all critical/high severity automatically dropped. FortiGuard updated daily.
Web Filter ACTIVE
FortiGuard category-based filtering. Blocks: Malware, Phishing, P2P, Gambling, Adult on Users VLAN.
App Control ACTIVE
L7 application awareness. Blocks: Tor, BitTorrent, Anonymous proxies. Monitors social media usage.
SSL Inspection DEEP
Deep SSL/TLS inspection on Users and Servers egress. Certificate pinning exceptions for banking sites.
Antivirus ACTIVE
FortiGuard AV scanning on HTTP/HTTPS/FTP/SMTP flows. Grayware detection enabled.
DNS Filter ACTIVE
FortiGuard DNS filtering. Blocks DNS-over-HTTPS bypass attempts. Sinkhole for known C2 domains.

CLI — DHCP Servers per VLAN

FG60E (server) # edit 1
    set interface     "vlan20"
    set dns-server1   1.1.1.1
    set dns-server2   8.8.8.8
    set default-gateway 192.168.20.1
    set netmask       255.255.255.0
    set lease-time    86400
    config ip-range

            set start-ip 192.168.20.100
            set end-ip   192.168.20.200
        next
    end
