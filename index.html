Enterprise-Style Security Lab

This project documents a practical network security home lab built around FortiGate and Cisco CBS250. It demonstrates VLAN segmentation, firewall policy enforcement, management plane protection, attack path analysis, troubleshooting methodology and SOC-oriented monitoring design.
