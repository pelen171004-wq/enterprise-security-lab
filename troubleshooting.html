Methodology

  • Verify IP address, mask, gateway and DNS
  • Validate VLAN assignment on switch access ports
  • Check inter-VLAN firewall policy path
  • Use FortiGate debug and packet analysis tools
  • Confirm routing, NAT and local-in policy behavior

Common Validation Commands

ping 10.10.20.1 ping 10.10.30.1 ping 1.1.1.1 nslookup google.com

FortiGate Debug Flow

diagnose debug reset diagnose debug flow filter addr 10.10.20.100 diagnose debug flow show console enable diagnose debug enable

Used to inspect firewall decisions, route lookup, NAT and policy match behavior.

Session Table

diagnose sys session filter src 10.10.20.100 diagnose sys session list

Used to verify active sessions, NAT translation and policy usage.

Packet Capture

diagnose sniffer packet any "host 10.10.20.100" 4

Used to confirm whether packets enter and leave expected interfaces.

Typical Issues Observed

  • Wrong DHCP or DNS values on client
  • Incorrect switch port VLAN assignment
  • Missing or misplaced firewall policy
  • Local-in policy blocking management access
  • Policy deny caused by inter-zone restrictions
